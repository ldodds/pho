Talis Platform Client

Pho is a Ruby client for the Talis Platform API. The API provides access to a semantic web data store capable of storing unstructured data (i.e. binary streams) as well as structured data (as RDF).

Pho provides a client-side API suitable for working with the Platform API. The code is largely a thin wrapper around the underlying HTTP calls, as well as a set of convenience classes and methods for parsing and working with responses from the Platform. The goal is to provide a complete API that also deals with RDF updates and mapping to/from Ruby objects.

Currently the best source of documentation is the rdocs.

There are also some blog posts on the Platform developer blog that illustrate use of the Pho API and command-line client.

As of version 0.7.4 the source code is managed in github.
